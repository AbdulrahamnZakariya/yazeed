Arduino IDE and Programming Basics

During the lecture, we also downloaded the Arduino IDE, which is the official software used for writing and uploading code to Arduino boards. The installation process was straightforward, and the IDE provides a simple interface to write, compile, and upload code to the Arduino.

We started by learning the basics of Arduino programming, such as setting up a simple program called a "sketch". The structure of an Arduino sketch includes two main functions:

  • void setup() – This function runs once when the Arduino board is powered on. It is used to initialize variables, pin modes, and libraries.
  • void loop() – After the setup, the loop function runs continuously. It is used to repeatedly execute a set of instructions, making it ideal for handling tasks that need to be checked or performed constantly.

We practiced simple commands, such as making an LED blink using the digitalWrite() and delay() functions, which are fundamental in controlling electronic components through code.

Understanding digitalWrite(), digitalRead(), analogWrite(), and analogRead()

In Arduino programming, we frequently use these four functions to interact with digital and analog pins. Here's the difference between them:

digitalWrite()

This function is used to set the state of a digital pin to either HIGH (5V) or LOW (0V). It is typically used to control components like LEDs or relays. 


digitalWrite(pin, HIGH);  // Turns on the component connected to the pin
digitalWrite(pin, LOW);   // Turns off the component connected to the pin

digitalRead()

This function reads the current state of a digital pin (either HIGH or LOW). It is used to read input from components like buttons or sensors that provide digital signals. 


int state = digitalRead(pin);  // Reads whether the pin is HIGH or LOW

analogWrite()

This function outputs a PWM (Pulse Width Modulation) signal on a pin, which can simulate an analog signal. It’s used for controlling components like LEDs with varying brightness or motors with varying speed. 


analogWrite(pin, value);  // Sends a PWM signal with a value between 0 (off) and 255 (full on)

analogRead()

This function reads the value from an analog input pin, which can be a value between 0 and 1023. It's often used to read sensors that provide varying voltage levels, such as potentiometers or temperature sensors. 


int sensorValue = analogRead(pin);  // Reads an analog value between 0 and 1023

Tinkercad Introduction

In addition to Arduino, we were introduced to Tinkercad, an online platform where we can simulate electronic circuits. We used Tinkercad to create virtual Arduino circuits and test different components like LEDs, resistors, and motors. This platform is very useful for testing designs before implementing them physically. It's easy to use and allows us to make adjustments quickly.

Through Tinkercad, we simulated a complete circuit where we connected sensors and controlled them through Arduino code. This made the process of building and testing circuits much more efficient, especially for beginners.


What is an Ultrasonic Sensor?

An ultrasonic sensor uses high-frequency sound waves to detect objects and measure distances. It works by emitting ultrasonic waves and measuring the time it takes for the echo to return.

Common Uses

  • Distance measurement for robots and automation systems.
  • Obstacle avoidance in autonomous vehicles.
  • Monitoring liquid levels in tanks.
  • Security systems for motion detection.
  • Car parking assistance systems.

What is a PIR Sensor?

A PIR (Passive Infrared) sensor detects infrared radiation emitted by warm objects, such as humans or animals, and is commonly used in security systems and motion-activated lighting.

Common Uses

  • Security alarms to detect intruders.
  • Automatic lighting systems in homes and public spaces.
  • Home automation to control devices based on occupancy.
  • Energy-saving systems to reduce power consumption.
  • Interactive games and entertainment systems.




What is a Soil Sensor?

A soil sensor measures various soil properties such as moisture, temperature, and pH levels. It's used in smart irrigation systems, agriculture, and environmental studies to monitor soil conditions.

Common Uses

  • Smart irrigation systems for optimal watering.
  • Precision farming to manage crops more efficiently.
  • Home and commercial garden monitoring.
  • Environmental research and studies.
  • Greenhouse automation systems.


LCD (Liquid Crystal Display) - Arduino and Potentiometer

What is an LCD?

LCD stands for Liquid Crystal Display. It is used in devices like TVs, monitors, and portable gadgets, working by aligning liquid crystals to block light and create images.

Common Uses

  • Digital clocks and watches
  • TVs and monitors
  • Calculators, handheld devices

Pros

  • Energy-efficient and bright
  • Lightweight and available in many sizes

Cons

  • Limited viewing angles
  • Lower refresh rates

Arduino Integration

You can connect an LCD to an Arduino using the LiquidCrystal library. Additionally, a potentiometer helps adjust screen contrast.

RGB LED - Overview and Uses

What is an RGB LED?

RGB LED is a type of light-emitting diode that combines red, green, and blue LEDs in one package. By adjusting the intensity of each color, it can produce a wide range of colors.

Common Uses

  • Decorative lighting in homes and events
  • Color-changing lamps
  • LED displays and signage
  • Indicating system states in electronic devices

Pros

  • Energy-efficient and long-lasting
  • Ability to produce millions of colors
  • Compact and easy to integrate into projects

Arduino Integration

RGB LEDs can be controlled using Arduino by adjusting the PWM (Pulse Width Modulation) signals on its red, green, and blue pins, allowing for dynamic color changes.

Joystick - Overview and Uses

What is a Joystick?

A joystick is an input device that allows control of movement in two axes: X (horizontal) and Y (vertical). It often includes a button for additional functionality and is commonly used in gaming and robotics.

Common Uses

  • Control systems in video games
  • Robotics and automation control
  • Remote-controlled vehicles and drones
  • Camera gimbal systems

Pros

  • Precise control in multiple directions
  • Intuitive and easy to use
  • Compact and widely available

Arduino Integration

Joysticks can be connected to Arduino by reading analog values from the X and Y axes, allowing for control of motors, servos, or other devices in interactive projects.



Conclusion

Joysticks are versatile input devices used in a variety of applications from gaming to robotics, offering precision control and ease of integration with microcontroller projects like Arduino.

Keypad - Overview and Uses

What is a Keypad?

A keypad is a small input device consisting of multiple buttons (keys), typically arranged in rows and columns. It is commonly used to input numbers, characters, or commands into a system.

Common Uses

  • Security systems (e.g., entering PINs)
  • Home appliances (e.g., microwaves, washing machines)
  • Automated access control systems
  • Numerical input for calculators and other devices

Pros

  • Simple and intuitive input method
  • Reliable for numeric and character inputs
  • Compact and easy to integrate into projects

Arduino Integration

Keypads can be connected to Arduino using a matrix configuration, where each button is assigned to a row and column. The Keypad library helps in detecting key presses and processing inputs efficiently.

Conclusion

Keypads are commonly used in systems that require numeric or character input, offering a straightforward and reliable way to interface with electronic devices.
