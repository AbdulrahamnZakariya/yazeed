SSBP

Smart Switch Button Pusher



Project Introduction

The "Smart Switch Button" project aims to design a remote-controlled smart switch for electrical devices,
operated via a mobile app or web interface.
The project utilizes Internet of Things (IoT) technology to enable device control from anywhere,
offering convenience and improving energy efficiency.

Project Objectives:

  • Remote control of electrical devices.
  • Enhanced energy consumption management.
  • Increased safety in smart homes.
  • User-friendly interface.

Short Explain about Components Used

The "Smart Switch Button" project utilizes the following components:

  • Servo Motor: Used to physically control the switch mechanism,
    allowing for the movement required to turn devices on or off.
  • ESP Controller (Wi-Fi & Bluetooth): A microcontroller that connects to both Wi-Fi and Bluetooth,
    enabling remote control of the switch via a mobile app or web interface.
  • 9V Battery: Provides the necessary power supply to the system,
    ensuring consistent operation of the servo motor and controller.
  • Touch Sensor: Acts as an input method,
    allowing users to manually control the switch with a simple touch.
