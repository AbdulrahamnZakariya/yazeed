Introduction to Web Development

Web development is the process of creating websites and applications that run on the web. It involves the use of various technologies and tools to build interactive and functional web pages. Two of the most foundational languages for building web pages are:

  • HTML (Hypertext Markup Language) - The structure of web content
  • CSS (Cascading Style Sheets) - The visual style and layout of web pages

During my assignments, I learned how to create websites using HTML and CSS, and experimented with different tools like Brackets, a lightweight text editor, and W3Schools, a learning platform for web technologies. These resources helped me understand the basics and get hands-on experience with web design.

HTML Basics

HTML serves as the backbone of any website, providing the structural layout of the page. In my learning, I created several web pages that demonstrate my understanding of various HTML elements such as:

  • Headers (h1 to h6)
  • Paragraphs, lists, links, and images
  • Forms and input elements

CSS Basics and Styling

CSS is what gives the web page its style and look. It controls things like layout, color, font, and more. One of the interesting topics I learned was the difference between the three types of CSS implementations:

  • Inline CSS: Styles applied directly within an HTML element.
  • Internal CSS: Styles applied within a single HTML file, using the <style> tag.
  • External CSS: A separate CSS file linked to the HTML document, allowing for better organization and reusability.
